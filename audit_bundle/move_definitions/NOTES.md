# Move Definitions Notes

- `src_data_moves.ts`: canonical move/exercise library, derived phase metadata, ground mode, camera preset, animator key mapping.
- `src_pages_Move.tsx`: move selection by slug and wiring into runtime viewer (`Viewer3D`).
- `src_pages_Library.tsx`: catalog/search layer that routes to move pages.
- `src_data_characters.ts`: character model definitions and per-character model path/scale defaults.
